### Key Features

* Internship Listings: A comprehensive database of internships from various companies and industries.
* Student Profiles: A platform for students to create detailed profiles showcasing their skills and experience.
* Application Management: Students can easily apply to internships and track the status of their applications.
* Company Dashboard: A dashboard for companies to post internships, review applications, and manage their internship program.

### Technologies Used

* Frontend: React.js, JavaScript, CSS ,Redux Toolkit
* Backend: Node.js, Express.js
* Database: MongoDB

### Running the Project
#### 1.Navigate to the Project Directory:
```bash
  cd InterniFy
```

#### 2.Installing Dependencies
- Client Side
  ```bash
  cd client
  npm i
  ```
  - Client Side
   ```bash
  cd server
  npm i
  ```

#### 3.Starting the development server
- Client Side
  ```bash
  cd client
  npm run dev
  ```
  - Client Side
   ```bash
  cd server
  nodemon
  ```
